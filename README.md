# 🎮 Frontend - Pokédex App

Aplicación web desarrollada con React que consume la PokéAPI y un backend propio para autenticación y favoritos.

## 🚀 Tecnologías

- React
- React Router
- CSS personalizado (estilo Pokédex)
- Fetch API

## 🔥 Funcionalidades

- Login / Registro
- Rutas protegidas
- Pokédex con paginación
- Buscador de Pokémon
- Detalle de Pokémon
- Añadir / eliminar favoritos en tiempo real

```bash
git clone <repo>
cd frontend
npm install
